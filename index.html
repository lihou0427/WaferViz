Wafer Visualization App - WaferViz

Introduction

WaferViz is an open-source wafer data plotting software intended for semiconductor engineering community.

Metrology tool vendors have their own plotting programs embedded in the tools while some other apps have been used around are either license restricted or unavailable to general public.

As a process engineer the author found it is very helpful having a program handy for converting raw wafer data to presentable graphs especially in fabs where plotting program is not readily accessible.
